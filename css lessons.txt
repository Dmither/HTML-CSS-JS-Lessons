Иерархия присвоения:
div{} - по тегу (для всех div)
.cla{} - по классу
div.cla{} - для всех div с классом class1
.cla span{} - для всех span внутри .class1
.cla>span{} - для span на первом уровне внутри .class1
Вложенные элементы наследуют некоторые параметры родителя

font-size: - размер текста
color: - цвет текста
нижние параметры перезаписывают верхние
