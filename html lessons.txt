Строительные теги, блочные (занимают всю ширину):
<div> - универсальный строительный блок
<header> - для шапки
<footer> - для подвала
<nav> - для организации навигации и меню

<H1> - <H6> - теги заголовка большего к меньшему (блочные)
<H1> используется в документе один раз, остальные любое количество
<p> - абзац (блочный)
<span> - выделение части текста (строчный)
Внутри строчных блоков не может быть блочных

<a href=""> - ссылка (на что либо) (строчный), href="" указывает ссылку
target="_blank" откроет в новом окне

<img src="" alt=""> изображение (строчный), путь и альтернативный текст

<ul> - маркированный список (блочный)
<ol> - нумерованный список (блочный)
<li> - пункт списка (блочный)

<br> - перевод на повую строку (строчный)

<form action=""> - контейнер для формы (блочный)
method="" указывает на метод передачи данных (get/post),
enctype="multipart/form-data" для отправки файлов с post
<input type=""> - поле формы (строчный), тип поля
(name="" - для обозначения имени поля при передачи на сервер
 value="" - значение по умолчанию
 plaseholder="" - подсказка значения text, textarea
 disabled - заблокированный
 tabindex - индекс при нажатии tab):
"text" - текстовое поле
"tel" - текстовое поле, на моб тел цифровая клавиатура
"password" - текстовое поле с сокрытием содержимого ***
"radio" - радиокнопки (один вариант выбора) (одно название и табиндекс)
"checkbox" - чекбокс (много вариантов выбора)
<textarea> - поле для передачи многострочного текста
<select name=""> - выпадающий список, имя, multiply - множ. выбор
<option value=""> - пункт выпадающего списка, selected - по умолч.
<button type=""> - кнопка. Тип: submit - отправить, reset - очистить
